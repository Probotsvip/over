YouTube API Service

High-performance YouTube content retrieval with MongoDB and Telegram caching

API v1.0.0

API Endpoints

GET /youtube

Retrieve YouTube video information and stream URLs with intelligent caching.

Parameters:
  • query - YouTube URL, video ID, or search term (required)
  • video - Boolean to get video stream (default: false for audio)
  • api_key - Your API key (required)
Example Request:
GET /youtube?query=https://youtube.com/watch?v=dQw4w9WgXcQ&video=true&api_key=your_key
Response Format:
{
  "status": true,
  "title": "Video Title",
  "duration": "Duration",
  "quality": "Quality",
  "url": "Real Download URL",
  "creator": "API Creator",
  "telegram": "@Telegram_Channel"
}
GET /health

Health check endpoint to verify API status.

Example Response:
{
  "status": "healthy",
  "version": "1.0.0",
  "timestamp": "2025-08-16T12:00:00Z"
}

Quick Start

                    

                        
Python Example:

                        
import httpx

async def get_youtube_stream(query):
    api_url = "http://your-api.com/youtube"
    params = {
        "query": query,
        "video": False,
        "api_key": "your_api_key"
    }
    
    async with httpx.AsyncClient() as client:
        response = await client.get(api_url, params=params)
        return response.json()
